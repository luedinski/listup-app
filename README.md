# Listup 🛒

> A collaborative shopping list app for desktop and mobile. Create, share, and manage lists together in real time — simple, clear, and always in sync.

---

## Features

- **Collaborative lists** – Invite others to a list and edit together in real time
- **Cross-platform** – Works seamlessly on desktop and mobile
- **Simple & clear** – Clean interface designed for fast, intuitive use
- **Real-time sync** – Changes appear instantly for all members
- **Organized overview** – All your lists in one place, always up to date

---

## Getting Started

### Prerequisites

- Node.js >= 18
- npm or yarn

### Installation

```bash
git clone https://github.com/your-username/listup.git
cd listup
npm install
```

### Run in development

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
